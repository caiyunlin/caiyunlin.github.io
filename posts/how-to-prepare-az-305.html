如何准备 AZ-305 考试

2024-08-29    azure

AZ-305 是微软提供的 Azure 认证考试之一，考试的全称是 Designing Microsoft Azure Infrastructure Solutions, 考试通过后获取的证书名称是 Microsoft Certified: Azure Solutions Architect Expert。

AZ-305 认证主要面向云解决方案架构师，其核心作用是帮助 IT 专业人士掌握在 Microsoft Azure 平台上设计和构建企业级云解决方案的能力，对于想从事云解决方案架构师的工作还是蛮有帮助的。

下面我列举一些自己在准备 AZ-305 考试的一些心得，仅供参考。

前置条件 ​

AZ-305 认证适合有一定 Azure 经验的专业人士，要考取 AZ-305 ，你需要有 Azure Administrator 相关的技能，必须先考取 AZ-104 ，也就是 Microsoft Certified: Azure Administrator Associate

考题形式 ​

微软AZ-305考试由40-60个问题组成。基于场景的单选题、多选题、顺序排列型题、拖放型题 以及 标注复习、拖放型题均包含在考试中。考生必须获得700分或更高的分数才能通过考试。因为相关技术的术语翻译可能不便于理解，我选择的是英文版本，AZ-305 考试的英文版本，费用为165美元。

考试范围 ​

考试的内容分为4大块，可以参考比率，详情可以在官网 https://learn.microsoft.com/en-us/credentials/certifications/exams/az-305/ 查看。

  • Design identity, governance, and monitoring solutions (25–30%)
  • Design data storage solutions (20–25%)
  • Design business continuity solutions (15–20%)
  • Design infrastructure solutions (30–35%)
  1. 建议是按照官网的 Self-paced learning path 去逐步学习，学习过程中会有一些 lab 训练，如果时间有限，可以先从概念上理解，后面有时间再补充lab。
  2. 学习完 learning path 之后，就可以准备考试了，注意考前一定有要刷几遍模拟考题，官网建议是超过80分再预约考试，我个人建议是最好多次超过90分再去预约。
  3. 对于整个概念的理解，我在文章底部也给出了 Youtube 视频讲解，我个人感觉还是蛮有用的，考前刷两遍，便于整体上有很好的总结和理解。
  4. 掌握好 learning path 和 Youtube 的两个视频，应该足够应付考试了，应该不需要再付费参加其他培训。

考前准备 ​

  • 报名：点击 官网 中的 "Schedule exam" 按钮，系统会跳转到 "Pearson Vue" 准备界面，如果你没有登录，则需提前登录 Microsoft 账号(@msn.com,@hotmail.com 或者 @live.com)之类，建议不要使用你的公司账号，因为证书是和邮箱绑定的。
  • 填写好你个人的联系方式，点击按钮跳转后，可以选择 "Online with OnVue", 这样系统会安排一个在线考试
  • 对于在线考试，需要注意： 1. 需要准备好电脑（考试前会收到邮件让运行一个 system test，需要通过测试)，2. 准备一个独立的空间（考试的时候，不要有其他人干扰，交谈，或者其他人出现在摄像头中），3. 身份证，4. 提前30分钟登录，以准备好考前验证之类的工作
  • 一路点击 OnVue 的 Next 页面，选择考试的日期和时间，最后会来到付款页面，绑定信用卡可以付款，如果公司有折扣的会在这一步自动抵扣，比如微软的员工就可以抵扣全额，直接提交即可。
  • OnVue会发送邮件到邮箱，提示考试日期以前需要提前准备的内容，有个模拟考试的程序，建议考前使用一下，这样真实考试的时候，也不会太陌生。

考试过程 ​

  • 提前30分钟登录后，运行 OnVue 程序
  • OnVue程序会关闭系统其他软件，验证麦克风和摄像头，考试过程中摄像头是全程开启的
  • 然后系统会提示你用手机扫描二维码，拍摄身份证，拍摄前后左右区域
  • 监考官会验证你的证件和区域，当验证完毕之后，就开始准备考试了，点击确定之后，你就全程不可以离开摄像头了
  • 然后就是答题，注意真实的考题和模拟考题题型是有不同的，模拟考试一般都是单选，但是真实考试有单选题、多选题、顺序排列型题、拖放型题，其中最开始还有场景模拟题，会设置一个基础的客户环境的场景，然后有一组题目，注意这组题目确认提交之后，是不可以再回去修改的。
  • 考试通过的分数是 700 分，在你提交的当时就会立刻显示你考试的分数，通知你有没有考试通过，如果没有通过也没有关系，系统会提示出哪一项你还比较薄弱，可以针对性的再准备，然后24小时候可以再次预约考试，考试有4次的重考机会。
